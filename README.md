## 🚀 주요 경험

### 🧠 AI 기반 정신건강 분석 시스템

qEEG Brain Mapping 데이터를 기반으로 정신건강 상태를 분류하고 분석하는  
AI 모델 및 서비스 시스템 개발에 참여했습니다.

- Python FastAPI 기반 백엔드 개발
- React / Next.js 기반 프론트엔드 개발
- DeepCluster, SimCLR, ResNet, ViT 기반 실험
- KMeans, UMAP, Silhouette Score를 활용한 클러스터링 평가
- 의료진과 협업하여 실제 진료 환경에 필요한 기능 구조화

### ⚖️ 법무관리 시스템 고도화

LG전자 법무관리 시스템 개편 프로젝트에서  
업무 흐름 중심의 프론트엔드 화면 개발에 참여했습니다.

- React 기반 엔터프라이즈 웹 개발
- 대시보드, 업무함, 법무 프로세스 중심 UI 구현
- 실제 사용자 업무 흐름을 고려한 화면 구조 설계
- 검색 및 데이터 조회 성능 개선 경험

### 🔐 Linux 기반 망분리 보안 시스템

Gooroom OS 기반의 망분리 보안 시스템 개발에 참여하며  
Linux 시스템 환경과 보안 구조를 다뤘습니다.

- Debian 기반 Gooroom OS 환경 개발
- PAM 인증 모듈 커스터마이징
- systemd 서비스 구성
- VirtualBox 기반 외부망/내부망 분리 구조 이해
- MariaDB Replication 및 Proxy 구성 경험
